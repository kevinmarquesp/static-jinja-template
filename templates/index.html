Embrace Simplicity

Writing a small project and React, Vue, Angular is too overkill for it? Regect modernity, return to plain HTML.

Consider using the TailwindCSS CDN for the styles and/or Alpine JS to handle the application's state; if you want simple UI library to speedup your development, I personaly recomend Pines UI for that. You can also check HTMX with the client-side-templatesextension to deal with REST APIs.

Finally, if you're having some problems with the templating system language checkout the Jinja2 docs for more information about for loops, macros, etc.
